# TypeFlow public showcase

This repo is the public face of TypeFlow.

The app itself is live here: [typefloww.vercel.app](https://typefloww.vercel.app)

The source code is private. That is intentional.

## What this repository is for

- Share what the product does
- Publish support and policy documents
- Keep a public update trail without exposing private implementation details

## Product summary

TypeFlow is a real-time typing product focused on speed, accuracy, and competitive play. It includes solo sessions, multiplayer matches, and performance tracking.
