# TypeFlow - Public Showcase

TypeFlow is a real-time multiplayer typing web application focused on speed, accuracy, and competitive arcade-style gameplay.

Important note:
- This repository is a public showcase only.
- The production source code is maintained in a private repository.

## Live Website

- https://typefloww.vercel.app

## What TypeFlow Includes

- Real-time multiplayer typing matches
- Competitive scoring and leaderboard systems
- Session tracking and player progress metrics
- Modern full-stack architecture optimized for web performance

## Why This Repo Exists

This repository helps demonstrate product work publicly while protecting implementation details, internal architecture, and private service logic.
