# Public Roadmap

This roadmap reflects public product direction and may evolve based on quality, security, and user feedback priorities.

## Near-Term Priorities

- Improve onboarding for first-time users
- Expand in-product performance insights
- Tune multiplayer reliability under variable network conditions
- Continue UX refinements for mobile and desktop consistency

## Mid-Term Priorities

- Broaden ranking and progression visibility
- Add more customization controls for typing sessions
- Improve diagnostics for faster issue triage
- Strengthen anti-abuse and fair-play controls

## Governance Notes

- Security and reliability work can reprioritize feature timelines.
- Roadmap items are directional, not contractual delivery guarantees.
