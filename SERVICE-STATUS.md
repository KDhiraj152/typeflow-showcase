# Service Status and Operations

TypeFlow is operated as a managed web service.

## Production Endpoint

- [typefloww.vercel.app](https://typefloww.vercel.app)

## Operating Model

- source code repository is private
- public showcase repository provides user-facing policy and support docs
- incidents and support requests are triaged through GitHub issues

## Incident Priority Model

- P1: outage or severe service-impacting issue
- P2: major functionality impaired
- P3: partial degradation with workaround
- P4: minor defect or informational issue

## Public Communication

When significant service-impacting incidents occur, updates are posted in relevant GitHub issues and reflected in documentation updates when needed.
